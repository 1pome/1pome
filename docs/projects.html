Workwave

Proyecto grupal

Workwave es una aplicación web desarrollada por un equipo multidisciplinario para la organización NoCountry Tech. Se nos desafió a hacer un clon de LinkedIn, pero, luego de lograr un consenso, decidimos darle nuestra impronta y crear nuestra propia versión de dicha red.

Desempeñé mi rol como parte del equipo backend, poniendo en práctica mis conocimientos de Django y Django REST Framework. Entre mis varias tareas, me encargué de co-crear el modelo entidad-relación para la base de datos, diseñar el sistema de registro y autenticación del sitio, y las operaciones CRUD para el perfil del usuario.

Fue una experiencia formidable donde puse a prueba mis conocimientos técnicos, pero más importante aún, mis habilidades interpersonales. La buena comunicación fue un factor clave para el éxito del grupo. No faltaron inconvenientes, desafíos y contratiempos, pero pudimos sortear todos estos obstáculos mediante el compromiso y diligencia de cada miembro.

Actualmente nos encontramos corrigiendo bugs, refactorizando algunas piezas de código y desarrollando nuevas funcionalidades para el sitio. Estamos muy ansiosos de ir compartiendo los progresos que vayamos logrando. Si te interesa conocer mi experiencia en detalle, te invito a contactarme a través de mis redes sociales o el formulario de esta misma página.
